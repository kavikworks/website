# Pricing Strategy

[[Kavik Works]] uses a productized SaaS model with one core offering and high-ticket add-ons.

## Current Pricing

| Product | Setup | Monthly |
|---------|-------|---------|
| Lead Response Automation (core) | $997 | $497/mo |
| Each add-on module | from $2,500 | +$400/mo |

## Design Principles

**1. Core looks cheap relative to add-ons.**
The $997/$497 core is designed to be an easy yes. Add-ons start at $2,500+ — high enough that Josh only takes them on when the ROI is unambiguous and the scope is clean.

**2. Monthly fee covers autonomy.**
$497/mo includes the running automation, monitoring, email support, and the automated monthly client report. No scheduled calls, no quarterly reviews. The fee is priced to be profitable without human time per client per month.

**3. Add-ons must justify custom scoping work.**
Cheap add-ons ($300/mo) create unpredictable scoping work without proportional return. At $2,500 setup + $400/mo, each module must have a clear ROI case to be worth pursuing.

**4. Setup fee covers implementation risk.**
$997 setup covers the cost of building, testing, and going live. Low enough to reduce friction, high enough to filter out uncommitted clients.

## Previous Model (Deprecated April 2026)

The old model used three tiers — Pilot/Standard/Growth — sold as a progressive roadmap. Problems:

- Required custom scoping for each client across multiple phases
- Unpredictable work to define what each phase meant per client
- Quarterly reviews baked in — not scalable beyond ~5 clients
- Pricing mismatch between website and intake form

## ROI Math for Clients

At 10+ leads/week, the ROI case is typically clear. Example:

- 10 leads/week × 30 min triage = 5 hrs/week = 20 hrs/month
- @ $35/hr labor cost = $700/month saved
- Retainer: $497/month
- **Net gain: $203/month + 21x lead qualification speed advantage**
- Setup payback: $997 / $203 ≈ 5 months
